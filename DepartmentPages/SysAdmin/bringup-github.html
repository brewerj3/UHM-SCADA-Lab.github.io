Github Tutorial

How to set up github

  1. Create a repo
    1. Go to profile
    2. Go to your repository
    3. click on new
    4. Modify to how you want. remember to add README file.
    5. If you haven't already make sure you create a token so you can log in to your account on pi.
      1. click on profile
      2. Go to developer settings
      3. Go to Personal Access tokens
      4. Click on tokens (classic)
      5. Generate a new token(MAKE SURE TO SAVE IT SOMEWHERE)

    How to set up your repo on pi

    1. Now that you set up your repo you want to clone it to the pi
    2. Make sure git is installed on pi(sudo apt install git)
    3. Add: git config --global user.name "Github_username"
    4. Add: git config --global user.email "Github_Email"
    5. git clone "git@github.com:repo-url"
      1. To clone from repo to pi on the repo click the green "<> code" button. Clone SSH and copy the link
    6. you may have to log in and input your token for your password
    7. you should have the repo on the pi now
    8. To make changes and push to repo
      1. git add . (adds all new changes)
      2. git status (this checks the new things being added/deleted)
      3. git commit -m "type any new changes here"
      4. git push (pushes all changes and commit to repo. You wil have to sign in and enter token)
      5. All new changes should be on github repo now
      6. If any new changes are added on repo but not on the pi
      7. git pull (pulls all new changes from the repo to pi)
